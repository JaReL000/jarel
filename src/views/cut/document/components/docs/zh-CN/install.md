## 安装


### NPM

推荐使用 NPM 安装`cube-ui`

```shell
$ npm install cube-ui
```


### CDN

也可通过 unpkg.com 获取资源，用标签引入 js 和 css 文件使用。

```html
<script src="https://unpkg.com/cube-ui/lib/cube.min.js"></script>
<link rel="stylesheet" href="https://unpkg.com/cube-ui/lib/cube.min.css">
```
